玩家交流条例

处罚力度​

  • 本规定适用于文档管理条例下所有条例分支中
  • 处罚力度会根据条例违反程度以及造成的影响而采取相应措施
等级处罚措施备注
轻度警告处理,如果警告三次无果上升为中度-
中度禁言1-10分钟,如果屡次触犯上升为重度-
重度禁言1天-1个月不等,如果多次触犯即上升为极重-
极重踢出群聊1天至不等，服务内封禁1天至不等,如果多次触犯即上升为BAN-
BAN永久拉黑用户,不得进入开Fun世界所有交流群聊以及所运营的服务器等-

群聊交流条例​

  1. 不得在群内发送淫秽，漏点以及其他违法国家规定，具有黄色情节的图片，视频等媒体文件和消息.

  2. 不得在群内发送血腥，暴力，猎奇等引人不适的文字，图片，音频，视频等媒体文件以及其他传播方式.

  3. 不得频繁发送长文字，重复和无用文字等具有刷屏性质的消息

  4. 不得频繁发送 "+1" 消息,每人仅限发送1次

  5. 尽量避免引战，进而发生不可调节的冲突。

  6. 禁止在未向服主申请前发布任何商业广告或宣传其他MC服务器 ，如有宣传需求请先私聊服主。

  7. 禁止使用淫梦梗(如114514、雪等)。

  8. 禁止发送带有明显攻击性的图片(特指龙图)、表情、文字等其他具有传播性质的文件和消息。

  9. 禁止使用第三方软件在开Fun世界群聊采用非法手段达成一些目的(如红包挂等)。

游戏交流条例​

  1. 不得在游戏内发送具有攻击性的消息

  2. 未经服主允许禁止向服务器发送具有宣传/商业性质的任何消息

  3. 禁止在游戏内发送大量长/重复/无意义等构成刷屏性质的消息

  4. 禁止发送可能造成服务器崩溃,干扰其他玩家正常聊天的消息
